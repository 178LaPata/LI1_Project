INTRODUÇÃO

Nesta tarefa, iremos tentar definir funções que nos permitam compactar um labirinto.

OBJETIVO

O nosso objetivo nesta tarefa é definir uma função que nos permita compactar o labirinto. Primeiro compactamos os corredores horizontalmente, depois compactamos verticalmente. Para cada corredor fizemos um elemIndices do corredor no labirinto, se o indíce do corredor fosse maior que o primeiro elemento do elemImdices então este corredor seria repetido do corredor com íncide igual ao do primeiro elemnto do elemIndices.

CONCLUSÃO

Concluindo conseguimos compactar o labirinto de uma forma eficiente.
