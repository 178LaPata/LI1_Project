Description

INTRODUÇÃO

Para esta tarefa o objetivo é a implementação de realização de jogadas para jogadores do tipo Pacman e Ghost

OBJETIVO

O objetivo desta tarefa é, dada uma descrição do estado do jogo e uma jogada de um dos jogadores, determinar o efeito dessa jogada no estado do jogo.

CONCLUSÃO

Conseguimos atualizar o efeito de uma jogada.
